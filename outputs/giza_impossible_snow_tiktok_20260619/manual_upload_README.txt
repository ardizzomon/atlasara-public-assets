History Vision manual upload package

Version 1 does not use unofficial browser login automation.
Use each platform folder's media file: video.mp4 for short-form video folders, pin_1.png for Pinterest, or slide images for carousel folders.

Suggested flow:
1. Open the platform folder.
2. Upload the media file named in that platform's posting_notes.txt.
3. Paste caption/hashtags/title files where relevant.
4. Add product URL where the platform supports links.

Product URL: https://beacons.ai/historyvision0
